## 🏗️ Architecture Overview

### Three MCP Servers (All Operational)
1. **code-analyzer** - Analyzes structure, dependencies, complexity (4 tools)
2. **git-analyzer** - Analyzes git history, contributors, hotspots (3 tools)
3. **documentation-generator** - Generates comprehensive docs (5 tools)

### Five AI Agents (Python)
1. **Architecture Analyzer** - Maps codebase structure
2. **Workflow Extractor** - Documents development processes
3. **Hotspot Detector** - Identifies technical debt
4. **Documentation Generator** - Creates onboarding guides
5. **Agent Coordinator** - Orchestrates all agents

## 🚀 Quick Start

### Prerequisites
- Python 3.9+
- Node.js 18+
- IBM Cloud account with watsonx.ai access

### Setup Steps
```bash
# 1. Install Python dependencies
pip install -r requirements.txt

# 2. Build MCP servers
cd src/mcp-servers && npm install && npm run build

# 3. Configure credentials
cp .env.example .env
# Edit .env with your WATSONX_API_KEY and WATSONX_PROJECT_ID

# 4. Run analysis
python run_analysis.py --repo-path ./test_repo
```
